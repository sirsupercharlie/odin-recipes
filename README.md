# Odin Recipes

This is my first HTML project from The Odin Project Foundations course.

The goal of this project is to build a basic recipe website using only HTML. The website will include a main index page with links to different recipes. Each recipe page will contain a title, an image, a description, ingredients, and step-by-step instructions.

By completing this project, I will have demonstrated basic HTML skills such as creating boilerplate structure, using headings, paragraphs, links, images, ordered and unordered lists, and organizing files correctly. I will also practice using Git and GitHub to manage and track my project.
